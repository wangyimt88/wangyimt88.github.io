快速创建属于自己的一个Hexo

前段时间自己摸索了搭建一个Hexo博客，发现还是挺有意思的，记录一波

安装node.js
直接去node.js官网下载

一个新版本，一个旧版本，点击就可以直接下载，然后按操作安装即可。

安装git
MAC好像是自带git的，只要能使用git指令就可以了。如果没有，去官网下载吧！

注册Github账号
想必大家都有自己的账号了，没有的话点击这里，去官网注册

创建仓库

点击New repository进行仓库的创建

Repository name填写格式:XXX.github.io （XXX填你的Owner里面的名称）
其他的全部默认就好。点击Create repository

添加秘钥
终端输入：ssh-keygen -t rsa -C 邮箱地址
然后敲回车就好了，然后就会生成2个文件id_rsa和id_rsa.pub，打开id_rsa.pub，复制里面的所有内容，然后进这里设置秘钥


点击Add SSH Key即可

最后只剩安装Hexo了
